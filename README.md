# 🏥 SwasthAI – Smart Healthcare Chatbot

SwasthAI is a full-stack AI-powered healthcare chatbot built using Java and Spring Boot for the backend, 
Thymeleaf, HTML, and CSS for the frontend, and
PostgreSQL for database management, integrating Spring AI (Ollama) for intelligent symptom-based medical guidance. 
It analyzes user symptoms and provides preliminary disease guidance through a secure login system and chat-based interface.
---

# 🚀 Key Features
- Secure User Authentication (Spring Security + BCrypt)
- Role-Based Authorization
- AI-powered Symptom Analysis (Spring AI - Ollama)
- PostgreSQL Database Integration
- Medical History Storage
- Responsive Chat UI (Thymeleaf)
---

# 🛠 Tech Stack

- Java 17
- Spring Boot
- Spring Security
- Spring Data JPA
- Spring AI (Ollama)
- PostgreSQL
- Thymeleaf
- Maven

---


 # ▶️ How to Run

# ▶️ How to Run

1. Configure PostgreSQL in `application.properties`.

2. (Optional) Set server port:
   server.port=9081
   If not specified, default port is 8080.

3. Run the application:
   mvn spring-boot:run

4. Access the application at:
   http://localhost:<server.port>
   Default:
   http://localhost:8080

---

# 🔐 Security

- Passwords encrypted using BCrypt
- Authenticated dashboard access
